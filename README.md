# Viagem no Tempo 🏛️

Landing page de agência de viagens com foco em civilizações antigas.
Desenvolvida como projeto prático do curso.

## Tecnologias
- HTML5
- CSS3
- JavaScript 

## Como visualizar
Abra o arquivo `index.html` no navegador
ou acesse a página: https://lokinarius.github.io/viagens--landing-page/

## Seções
- **Banner** — imagem de destaque com texto sobreposto
- **Viagem no Tempo** — cards dos destinos
- **Conselhos** — dicas de viagem
- **Nos Encontre** — contato e localização
